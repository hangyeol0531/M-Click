# M-Click(C#.Net WinForm)
GSM 반장/부장 학생 관리 시스템 
<br><br>

## 메인화면<br>
학생관리, 자리배치, 안내장 관리 폼으로 이동가능한 버튼 3개를 배치하고 왼쪽에는 학생들의 정보를 확인할 수 있는 Datagridview를 배치하였습니다.

## 학생 관리<br>
학생들의 정보들을 textbox와 button을 통하여 검색, 추가 , 수정 , 삭제를 제작해보았습니다.
